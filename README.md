</details>

<details>
<summary>вҡЎ <strong>Installation Guide</strong></summary>

### рҹҺҜ Step-by-Step Installation

**Step 1: Clone Repository**
```bash
# Clone the project
git clone https://github.com/deepakstwt/NeighborFit.git
cd NeighborFit
```

**Step 2: Backend Setup**
```bash
# Navigate to backend
cd neighborfit/backend

# Install dependencies
npm install

# Create environment file
cp env.example .env

# Configure your .env file (see environment setup)
```

**Step 3: Frontend Setup**
```bash
# Open new terminal and navigate to frontend
cd neighborfit/frontend

# Install dependencies
npm install

# Create environment file
echo "REACT_APP_API_URL=http://localhost:5001/api" > .env
```

**Step 4: Start Development Servers**
```bash
# Terminal 1: Start backend
cd neighborfit/backend && npm start

# Terminal 2: Start frontend
cd neighborfit/frontend && npm start
```

**Step 5: Access Application**
- рҹҢҗ **Frontend:** http://localhost:3000
- рҹ”§ **Backend:** http://localhost:5001
- рҹ“Ҡ **API Docs:** http://localhost:5001/api

</details>

<details>
<summary>вҡҷпёҸ <strong>Environment Configuration</strong></summary>

### рҹ”§ Backend Environment Setup

**Create `.env` file in `neighborfit/backend/`:**
```env
# Database Configuration
MONGODB_URI=mongodb://localhost:27017/neighborfit

# Authentication
JWT_SECRET=your-super-secret-jwt-key-here

# Email Configuration
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-specific-password

# Server Configuration
PORT=5001
NODE_ENV=development
CLIENT_URL=http://localhost:3000
```

### рҹҺЁ Frontend Environment Setup

**Create `.env` file in `neighborfit/frontend/`:**
```env
# API Configuration
REACT_APP_API_URL=http://localhost:5001/api

# App Configuration
REACT_APP_NAME=NeighborFit
REACT_APP_VERSION=1.0.0
```

### рҹ“§ Email Setup Guide

**Gmail Configuration:**
1. Enable 2-factor authentication
2. Generate app-specific password
3. Use app password in EMAIL_PASS
4. Update EMAIL_USER with your Gmail

**Other Email Providers:**
- Outlook: Use SMTP settings
- Yahoo: Generate app password
- Custom SMTP: Configure host/port

```env
JWT_SECRET=your-production-jwt-secret
EMAIL_USER=your-production-email
EMAIL_PASS=your-production-email-password

```

**Step 4: Deploy**
- Click "Deploy"
- Wait for build completion
- Your app is live! рҹҺү
