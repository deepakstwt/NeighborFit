## 🚀 Quick Start

<details>
<summary>📋 <strong>System Requirements</strong></summary>

### ✅ Prerequisites Checklist

- [ ] **Node.js** (v16.0 or higher)
- [ ] **MongoDB** (local installation or Atlas account)
- [ ] **npm** (comes with Node.js)
- [ ] **Git** (for version control)
- [ ] **Code Editor** (VS Code recommended)

**System Compatibility:**
- ✅ Windows 10/11
- ✅ macOS 10.15+
- ✅ Linux (Ubuntu 18.04+)

</details>

<details>
<summary>⚡ <strong>Installation Guide</strong></summary>

### 🎯 Step-by-Step Installation

**Step 1: Clone Repository**
```bash
# Clone the project
git clone https://github.com/deepakstwt/NeighborFit.git
cd NeighborFit
```

**Step 2: Backend Setup**
```bash
# Navigate to backend
cd neighborfit/backend

# Install dependencies
npm install

# Create environment file
cp env.example .env

# Configure your .env file (see environment setup)
```

**Step 3: Frontend Setup**
```bash
# Open new terminal and navigate to frontend
cd neighborfit/frontend

# Install dependencies
npm install

# Create environment file
echo "REACT_APP_API_URL=http://localhost:5001/api" > .env
```

**Step 4: Start Development Servers**
```bash
# Terminal 1: Start backend
cd neighborfit/backend && npm start

# Terminal 2: Start frontend
cd neighborfit/frontend && npm start
```

**Step 5: Access Application**
- 🌐 **Frontend:** http://localhost:3000
- 🔧 **Backend:** http://localhost:5001
- 📊 **API Docs:** http://localhost:5001/api

</details>

<details>
<summary>⚙️ <strong>Environment Configuration</strong></summary>

### 🔧 Backend Environment Setup

**Create `.env` file in `neighborfit/backend/`:**
```env
# Database Configuration
MONGODB_URI=mongodb://localhost:27017/neighborfit

# Authentication
JWT_SECRET=your-super-secret-jwt-key-here

# Email Configuration
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-specific-password

# Server Configuration
PORT=5001
NODE_ENV=development
CLIENT_URL=http://localhost:3000
```

### 🎨 Frontend Environment Setup

**Create `.env` file in `neighborfit/frontend/`:**
```env
# API Configuration
REACT_APP_API_URL=http://localhost:5001/api

# App Configuration
REACT_APP_NAME=NeighborFit
REACT_APP_VERSION=1.0.0
```

### 📧 Email Setup Guide

**Gmail Configuration:**
1. Enable 2-factor authentication
2. Generate app-specific password
3. Use app password in EMAIL_PASS
4. Update EMAIL_USER with your Gmail

**Other Email Providers:**
- Outlook: Use SMTP settings
- Yahoo: Generate app password
- Custom SMTP: Configure host/port

</details>

<details>
<summary>🌐 <strong>Vercel Deployment Guide</strong></summary>

### 🎯 One-Click Deployment

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/deepakstwt/NeighborFit)

### 📋 Manual Deployment Steps

**Step 1: Prepare Repository**
```bash
# Ensure all changes are committed
git add .
git commit -m "Prepare for deployment"
git push origin main
```

**Step 2: Vercel Configuration**
1. Sign in to [Vercel](https://vercel.com)
2. Click "New Project"
3. Import your GitHub repository
4. Configure build settings:
   - Build Command: `npm run build`
   - Output Directory: `build`
   - Install Command: `npm install`

**Step 3: Environment Variables**
```env
# Add these in Vercel dashboard
MONGODB_URI=your-mongodb-atlas-connection-string
JWT_SECRET=your-production-jwt-secret
EMAIL_USER=your-production-email
EMAIL_PASS=your-production-email-password
NODE_ENV=production
```

**Step 4: Deploy**
- Click "Deploy"
- Wait for build completion
- Your app is live! 🎉

**Custom Domain:**
1. Add domain in Vercel dashboard
2. Configure DNS settings
